# CCTI

Coder Character Type Indicator is an MBTI-style coding personality test for developers.

Run it directly from the repository:

```bash
npx github:peter-xs/CCTI
```

Or locally:

```bash
npm start
```

## Phase 1

- Interactive CLI test
- 28-question starter question bank
- Four-dimensional scoring model
- 16 CCTI result types
- Text result report

## Phase 2

- Ink-based interactive TUI with arrow key navigation
- Terminal radar chart visualization
- ASCII share card

## Commands

```bash
ccti
ccti --json
ccti --answers AAAAAAAAAAAAAAAAAAAAAAAAAAAA
ccti --types
ccti --help
```

## Controls

- **Arrow Up/Down** or **A/B** or **1/2**: Select option
- **Enter**: Confirm answer
- **S** (on result page): View share card
- **Enter** (on result/share page): Exit
